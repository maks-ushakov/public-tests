#Ensure

##Description

Modify the _ensure_ function so that it throws an error if called without arguments or an argument is _undefined_. Otherwise it should return the given value.


```js
function ensure(value) {
  return value;
}
```

##Test Cases  
  * No arguments
  
  * Truthy argument
  
  * Falsy argument
